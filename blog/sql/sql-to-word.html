# PowerDesigner导出word

需要word模板，下载地址 https://pan.baidu.com/s/1UOKS8Y-H8XVOv7HDT2AOVw

# 1 导入项目

首先还是先打开我们的PowerDesigner,打开我们要导出的项目

# 2 进入Report模版

点击Report-->Reports templates 进入Report模版页面

# 3 打开模板

点击最右边的 文件夹图标,进入选择页面,找到我们下载的模版文件(我的在F:\mdt,到文件夹那一层就好),选好以后点击'确定',然后再点击'close'关闭

# 4 进入导出页面

点击Report-->Generate Reports 进入导出页面

# 5 选择导出格式

依照图片选择(备注:Generate html导出的是一个html格式的),还有就是不要忘记设置为简体中文

# 6 查看导出文件

点击ok或者确定 就可以导出文件了，弹出以下信息说明导出成功,打开查看导出文档
